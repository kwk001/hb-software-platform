## 一、页面整体风格定位

### 1.1 设计风格
- **风格类型**：政府政务风格 + 科技感
- **整体调性**：专业、权威、简洁、现代
- **视觉特征**：
  - 采用蓝色系为主色调，体现科技感和政府公信力
  - 大面积使用白色背景，保持页面清爽
  - 卡片式布局，信息层次清晰
  - 毛玻璃效果（Glassmorphism）在Banner区域的应用

### 1.2 页面结构
```
┌─────────────────────────────────────────────────────────┐
│  Header (Logo + 标题 + 日期 + 登录/注册)                  │
├─────────────────────────────────────────────────────────┤
│  Navigation (首页/申报服务/政策资讯/产品服务/人才培养/数据统计)│
├─────────────────────────────────────────────────────────┤
│  Hero Banner (主视觉区 + 数据统计 + CTA按钮)              │
├─────────────────────────────────────────────────────────┤
│  申报服务 (申报通知卡片列表)                              │
├─────────────────────────────────────────────────────────┤
│  申报流程 (8步骤流程图)                                  │
├─────────────────────────────────────────────────────────┤
│  政策资讯 (政策列表)                                     │
├─────────────────────────────────────────────────────────┤
│  产品服务 (6类软件产品展示)                              │
├─────────────────────────────────────────────────────────┤
│  人才培养 (培训/考试/案例)                               │
├─────────────────────────────────────────────────────────┤
│  Footer (在线咨询 + 联系方式)                            │
└─────────────────────────────────────────────────────────┘
```

## 二、色彩设计规范

### 2.1 主色调
| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| **主品牌蓝** | `#0052D9` / `rgb(0, 82, 217)` | Logo、主标题、重点文字、链接 |
| **深蓝** | `#1D2129` / `rgb(29, 33, 41)` | 正文标题、重要文字 |
| **科技蓝** | `#409EFF` / `rgb(64, 158, 255)` | Element UI 主色、按钮高亮 |

### 2.2 辅助色
| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| **成功绿** | `#00A870` / `rgb(0, 168, 112)` | 状态标签（申报中）、成功提示 |
| **浅绿背景** | `#E8F5E9` / `rgb(232, 245, 233)` | 成功状态背景 |
| **信息蓝** | `#E8F3FF` / `rgb(232, 243, 255)` | 信息提示背景 |
| **浅色背景** | `#F5F7FA` / `rgb(245, 247, 250)` | 页面背景、卡片背景 |

### 2.3 中性色
| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| **纯白** | `#FFFFFF` | 卡片背景、按钮文字 |
| **浅灰** | `#F7F8FA` / `rgb(247, 248, 250)` | 区块背景 |
| **边框灰** | `#E5E6EB` / `rgb(229, 230, 235)` | 边框、分割线 |
| **次要文字** | `#86909C` / `rgb(134, 144, 156)` | 辅助文字、描述文字 |
| **正文灰** | `#4E5969` / `rgb(78, 89, 105)` | 正文内容 |
| **深灰** | `#1D2129` / `rgb(29, 33, 41)` | 标题文字 |

### 2.4 毛玻璃效果色
| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| **半透明白** | `rgba(255, 255, 255, 0.15)` | 卡片背景 |
| **半透明白边框** | `rgba(255, 255, 255, 0.3)` | 卡片边框 |
| **半透明白文字** | `rgba(255, 255, 255, 0.9)` | Banner区文字 |
| **半透明白次级** | `rgba(255, 255, 255, 0.7)` | 次要信息 |

### 2.5 色彩应用原则
```
┌─────────────────────────────────────────────────────┐
│  背景色使用层级：                                      │
│  页面背景 → #F5F7FA (浅灰)                            │
│  区块背景 → #FFFFFF (纯白)                            │
│  Banner背景 → 渐变蓝色 + 图片叠加                      │
│                                                      │
│  文字色使用层级：                                      │
│  主标题 → #0052D9 (品牌蓝)                            │
│  次级标题 → #1D2129 (深灰)                            │
│  正文 → #4E5969 (中灰)                                │
│  辅助文字 → #86909C (浅灰)                            │
│                                                      │
│  状态色：                                             │
│  成功/申报中 → #00A870 (绿色)                         │
│  主要操作 → #409EFF (蓝色)                            │
└─────────────────────────────────────────────────────┘
```

## 三、字体排版规范

### 3.1 字体家族
```css
font-family: "Microsoft YaHei", "Helvetica Neue", Helvetica, 
             "PingFang SC", "Hiragino Sans GB", Arial, sans-serif;
```

### 3.2 字体层级
| 层级 | 字号 | 字重 | 行高 | 用途 |
|-----|------|------|------|------|
| **H1** | 22px | 700 (Bold) | normal | 平台主标题 |
| **H2** | 32px | 600 (SemiBold) | 48px | 区块大标题 |
| **H3** | 24px | 600 (SemiBold) | normal | 卡片标题、小标题 |
| **H4** | 18px | 600 (SemiBold) | normal | 列表项标题 |
| **正文** | 16px | 400 (Regular) | 1.5 | 段落文字 |
| **小字** | 14px | 400 (Regular) | normal | 描述、辅助信息 |
| **标签** | 12px | 400 (Regular) | normal | 标签、时间 |
| **导航** | 15px | 600 (SemiBold) | normal | 导航菜单 |

### 3.3 排版原则
- **标题**：使用较深的颜色（#1D2129 或 #0052D9），字重 600-700
- **正文**：使用灰色（#4E5969），字重 400，行高 1.5-1.75
- **辅助文字**：使用浅灰（#86909C），字号 12-14px
- **链接**：使用品牌蓝（#0052D9），hover 时加深

## 八、动效规范

### 8.1 过渡动画
```css
/* 标准过渡 */
transition: all 0.3s cubic-bezier(0.23, 1, 0.32, 1);

/* 快速过渡 */
transition: all 0.2s ease;

/* 颜色过渡 */

```

### 8.2 交互反馈
| 交互 | 效果 | 时长 |
|-----|------|------|
| **按钮Hover** | 背景加深 + 轻微上移 | 200ms |
| **卡片Hover** | 阴影加深 + 上移2px | 300ms |
| **链接Hover** | 颜色变化 | 200ms |

## 十、设计原则总结

### 10.1 核心原则
1. **简洁清晰**：信息层级分明，留白充足
2. **专业权威**：蓝色主调，体现政府公信力
3. **用户友好**：操作流程清晰，反馈及时
4. **现代感**：毛玻璃效果、圆角、柔和阴影

### 10.2 设计建议
1. **保持一致性**：所有页面遵循同一套设计规范
2. **突出重点**：使用品牌蓝突出重要信息和操作
3. **合理留白**：避免页面拥挤，保持呼吸感
4. **响应优先**：确保各设备上的良好体验
