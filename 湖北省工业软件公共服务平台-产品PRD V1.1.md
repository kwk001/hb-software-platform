## 一、文档概述

### 1.1 背景
湖北省作为制造业大省，工业软件需求旺盛，但供需双方信息不对称、对接效率低。为促进工业软件产业发展，湖北省计划建设"工业软件公共服务平台"，连接工业软件企业（供给方）和工业制造企业（需求方），提供软件产品展示、补贴券申报、供需对接等一站式服务。

### 1.2 目标
- **连接供需双方**：建立工业软件企业与制造企业的对接桥梁
- **促进产业发展**：通过补贴券政策，降低企业数字化成本
- **提升服务效率**：线上化入驻、审核、申报流程，提高运营效率
- **数据驱动决策**：建立完善的报表体系，支持政策制定

### 1.3 适用范围
本文档适用于湖北省工业软件公共服务平台的全套产品功能设计。

## 二、产品定位与目标

### 2.1 产品定位
**湖北省工业软件公共服务平台**是一个连接工业软件供需双方的公共服务平台，旨在：
- 为制造企业提供工业软件产品发现和补贴申请服务
- 为软件企业提供产品展示和需求对接服务
- 为政府提供产业数据统计和政策发布渠道

### 2.2 核心价值

| 价值维度 | 核心价值 |
|---------|---------|
| **对制造企业** | 发现优质工业软件、申请补贴降低采购成本 |
| **对软件企业** | 展示产品、获取客户、提升品牌影响力 |
| **对政府** | 掌握产业数据、精准施策、推动产业发展 |

### 2.3 产品目标

| 目标类型 | 具体目标 |
|---------|---------|
| **目标** | 入驻企业100+，上架软件100+，发放补贴5000万 |

## 五、功能模块

### 5.1 企业入驻与审核模块

#### 5.1.1 功能概述
建立标准化的企业入驻流程，支持工业制造企业和工业软件企业入驻，实现线上化审核。

#### 5.1.2 核心功能

| 功能点 | 说明 | 使用角色 |
|--------|------|---------|
| 手机号注册 | 手机号+验证码注册 | 所有企业 |
| 入驻申请 | 填写企业信息、上传资质文件 | 所有企业 |
| 入驻审核 | 平台管理员审核企业资质（一级审核） | 平台管理员 |
| 入驻状态查询 | 查看审核进度和结果 | 所有企业 |
| 重新申请 | 审核驳回后修改重新提交 | 所有企业 |

#### 5.1.3 状态流转

| 状态码 | 状态名称 | 说明 |
|--------|----------|------|
| 0 | 已注册 | 完成手机号注册，未提交入驻申请 |
| 1 | 待审核 | 已提交入驻申请，等待平台审核 |
| 2 | 审核中 | 平台正在审核中 |
| 3 | 已入驻 | 审核通过，入驻成功 |
| 4 | 已驳回 | 审核未通过 |

#### 5.1.4 入驻字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 企业名称 | 输入框 | ✅ | 营业执照上的企业全称 |
| 统一社会信用代码 | 输入框 | ✅ | 18位信用代码 |
| 企业类型 | 单选 | ✅ | 制造企业/软件企业 |
| 所属行业 | 下拉选择 | ✅ | 从数据字典选择 |
| 企业规模 | 下拉选择 | ✅ | 50人以下/50-200人等 |
| 注册地址 | 级联选择 | ✅ | 省市区+详细地址 |
| 营业执照 | 文件上传 | ✅ | 图片或PDF |
| 联系人姓名 | 输入框 | ✅ | - |
| 联系人电话 | 输入框 | ✅ | - |
| 联系人邮箱 | 输入框 | ✅ | - |
