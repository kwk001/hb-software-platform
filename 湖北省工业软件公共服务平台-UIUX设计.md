## 一、设计概述

### 1.1 设计理念

结合**湖北经信工业申报平台**的专业政务风格与**湖北省工业软件公共服务平台**的业务特性，打造：

- **专业可信**：政府平台属性，蓝色主调传递权威感
- **简洁高效**：信息层次清晰，操作流程简化
- **现代科技**：毛玻璃效果、渐变、微动效提升体验
- **用户友好**：符合WCAG无障碍标准，响应式设计

### 1.2 设计目标

| 目标 | 说明 |
|------|------|
| **品牌一致性** | 与湖北经信平台保持视觉一致性，建立用户信任 |
| **业务匹配** | 针对B2B企业用户，设计专业、高效的操作界面 |
| **多端适配** | 支持桌面端、平板、手机多端访问 |
| **无障碍** | 符合WCAG 2.1 AA标准，支持键盘导航、屏幕阅读器 |

### 1.3 系统划分

| 系统 | 用户群体 | 主要功能 |
|------|---------|---------|
| **平台门户** | 访客、企业用户 | 软件展示、政策查看、需求浏览 |
| **企业端** | 制造企业、软件企业 | 入驻申请、软件发布、补贴申报 |
| **平台端** | 平台管理员、审核人员 | 审核管理、数据统计、系统配置 |

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│  注册    │───→│ 填写资料 │───→│ 提交审核 │───→│ 平台审核 │───→│ 入驻成功 │
│  手机号  │    │ 企业信息 │    │          │    │          │    │          │
└──────────┘    └──────────┘    └──────────┘    └────┬─────┘    └──────────┘
                                                      │
                                               ┌──────┴──────┐
                                               │             │
                                          ┌────┴────┐   ┌────┴────┐
                                          │ 审核通过 │   │ 审核驳回 │

                                                        ┌────┴────┐
                                                        │ 修改重提 │
                                                        └────┬────┘
                                                             │
                                                             └──────→ 提交审核
```

**页面流程：**
1. **注册页** → 输入手机号 → 获取验证码 → 设置密码 → 注册成功
2. **入驻申请页** → 填写企业信息 → 上传资质文件 → 提交申请
3. **等待审核页** → 显示审核进度 → 查看审核状态
4. **审核结果页** → 通过: 进入工作台 / 驳回: 查看原因 → 修改重提

## 九、动效设计规范

### 9.1 过渡动画

```css
/* 标准过渡 */
--transition-fast: 150ms ease;
--transition-normal: 200ms ease;
--transition-slow: 300ms ease;

/* 页面切换 */
.page-enter {
  opacity: 0;
  transform: translateY(10px);
}
.page-enter-active {
  opacity: 1;
  transform: translateY(0);
  transition: opacity 300ms ease, transform 300ms ease;
}

/* 卡片Hover */
.card {
  transition: box-shadow 200ms ease, transform 200ms ease;
}
.card:hover {
  box-shadow: 0 4px 16px rgba(0, 0, 0, 0.08);
  transform: translateY(-2px);
}

/* 按钮Hover */
.btn {
  transition: background-color 200ms ease, 
              border-color 200ms ease, 
              box-shadow 200ms ease;
}
```

### 9.2 加载状态

```css
/* 骨架屏 */
.skeleton {
  background: linear-gradient(
    90deg,
    #F2F3F5 25%,
    #E5E6EB 50%,
    #F2F3F5 75%
  );
  background-size: 200% 100%;
  animation: skeleton-loading 1.5s infinite;
}

@keyframes skeleton-loading {
  0% { background-position: 200% 0; }
  100% { background-position: -200% 0; }
}

/* 加载动画 */
.spinner {

  height: 24px;

}

  to { transform: rotate(360deg); }
}
```

## 十、无障碍设计规范

### 10.1 键盘导航

- 所有交互元素支持 Tab 键导航
- 焦点状态清晰可见（蓝色边框）
- Enter/Space 键触发按钮点击
- Esc 键关闭弹窗

```css
/* 焦点状态 */
:focus-visible {
  outline: 2px solid #0052D9;
  outline-offset: 2px;
}

/* 跳过链接 */
.skip-link {
  position: absolute;
  top: -40px;
  left: 0;
  background: #0052D9;
  color: white;
  padding: 8px;
  z-index: 100;
}
.skip-link:focus {
  top: 0;
}
```

### 10.2 屏幕阅读器

- 所有图片添加 alt 属性
- 表单元素关联 label
- 使用语义化 HTML 标签
- 动态内容使用 ARIA live regions

```html
<!-- 图片 -->
<img src="logo.png" alt="湖北省工业软件公共服务平台 Logo">

<!-- 表单 -->
<label for="company-name">企业名称</label>
<input id="company-name" type="text" aria-required="true">

<!-- 按钮 -->
<button aria-label="关闭弹窗">✕</button>

<!-- 状态提示 -->
<div role="status" aria-live="polite">
  提交成功
</div>
```
