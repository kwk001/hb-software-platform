## 图片规格建议

- 格式：SVG（推荐）或 PNG
- 尺寸：宽度 120-160px，高度 40-50px
- 背景：透明背景
- 颜色：建议使用企业标准色

## 注意事项

1. 确保已获得企业 Logo 使用授权
2. Logo 文件应保持清晰，避免变形
3. 建议统一风格（彩色或单色）
